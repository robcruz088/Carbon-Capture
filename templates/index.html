About the Project

This project started off at the Rutgers University-Camden in order to track the pollution levels around campus. The scope of the project has since then changed, and a prototype is in the works in order to track air pollutants wherever these Pollution Boxes are placed. Currently, the device can only track a few pollutants in the air, these being Carbon Monoxide and Particulate Matter. In order to track data effectively, the Temperature and Humidity is collected in order to check for any correlation between these and the Pollutant Levels. Planned Updates
